#+TITLE: chamkho-pg

chamkho-pg is a Rust port of [[https://github.com/zdk/pg-search-thai][pg-search-thai]]. It's a PostgreSQL extension whose objective is to enable full-text searching on Southeast Asian and other languages. Currently, chamkho-pg supports Chinese, Japanese, Khmer, Myanmar, Lao, Shan, Thai, and other space-delimited languages.

* How to install

** Prerequisites

- Rust toolchain (https://rustup.rs)
- pgrx CLI: ~cargo install cargo-pgrx~
- pgrx initialized: ~cargo pgrx init~

** Install

1. git clone https://github.com/veer66/chamkho-pg.git
2. cd chamkho-pg
3. ./install.sh

** Development

- ~cargo pgrx run~ — builds and starts a Postgres instance with the extension installed
- ~cargo pgrx test~ — runs the test suite

/Note:/ ~cargo pgrx install~ only installs the extension files (\.control, \.so, \.sql). The dictionary data in ~tsearch_data/~ is copied separately by ~install.sh~.

* Status

chamkho-pg uses [[https://github.com/pgcentralfoundation/pgrx][pgrx]] and currently supports PostgreSQL 15, 16, 17, and 18 on both GNU/Linux and macOS.

Current Docker images are built with PostgreSQL 18.

* Example

** Initialize

#+begin_src sql
  CREATE EXTENSION chamkho_parser;
  CREATE TEXT SEARCH CONFIGURATION chamkho (PARSER = chamkho_parser);
  ALTER TEXT SEARCH CONFIGURATION chamkho ADD MAPPING FOR word WITH simple;
#+end_src

** Prepare table

#+begin_src sql
  create table tab1(id serial, body text);
  insert into tab1(body) values ('ไก่กับเป็ด'), ('ช้างม้า'), ('วัวหมี');
#+end_src

** Query

#+begin_src sql
  select * from tab1 where to_tsvector('chamkho', body) @@ to_tsquery('เป็ด & ไก่');
#+end_src

** Index

#+begin_src sql
  CREATE INDEX tab1_idx ON tab1 USING GIN (to_tsvector('chamkho', body));
#+end_src

* Podman

** Build

#+begin_src sh
  $ git clone https://github.com/veer66/chamkho-pg.git
  $ cd chamkho-pg
  $ podman build -t chamkho-pg .
#+end_src

** Run

#+begin_src sh
  $ podman run --name chamkho-pg-1 -e POSTGRES_PASSWORD=yourpass -d chamkho-pg
#+end_src

** Use

#+begin_src sh
  $ podman exec -it chamkho-pg-1 psql -U postgres
#+end_src
